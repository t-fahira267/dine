# Models
- **[Google ViTs](https://github.com/google-research/vision_transformer)**:
  - Pre-trained
  - Model readily-available in Google Cloud

- **[EfficientNet-B0](https://huggingface.co/google/efficientnet-b0)**
  - Trained on ImageNet-1k
  - Resolution 224x224
  - [Achieved 97% accuracy](https://github.com/t-fahira267/dine/blob/master/notebooks/efficientnetb0_baseline.ipynb) with current MM-Food-100K dataset
